WildEntityRemover

Client-Side Mod for Minecraft 1.21.1

Overview

WildEntityRemover is a lightweight client-side mod designed to improve performance and overall gameplay experience by hiding unnecessary entities.

Originally developed for the WildNetwork, this mod helps reduce lag and increase FPS in entity-heavy environments.

Features

🚀 Boosts FPS by hiding entities

🎮 Fully client-side (no server required)

⚡ Improves stability in crowded areas

Purpose

This mod was created to enhance performance on servers like WildNetwork, where large numbers of entities can impact gameplay smoothness.
